How to use

1. Choose an image (PNG or JPEG, max 4500x4500 pixels)

2. Enter the text you want to render (max 10,000 characters)

3. Select render (per pixel) and color modes

4. Click "Render" and wait for the result

Tip #1: Use high-contrast and/or higher quality images for better results

Tip #2: Shorter text yields clearer images

Tip #3: Click on the image to expand, scroll to zoom

Tip #4: By character will generally be more detailed
